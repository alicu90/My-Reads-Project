# MyReads Project

This project will help you to manage your books which you read or you wish to do it in the future.

You can have a library of books in different phases of reading (ex: currently reading, want to read, read) and a search tool.

## How to open this project

To get started:

* install all project dependencies with `npm install`
* start the development server with `npm start`
* create production optimized build with `npm build`
* run tests with `npm test`
